##  Project Overview
This project demonstrates advanced Python fundamentals by implementing optimized algorithms and a CLI-based application.

---

##  Features

### 🔹 Level 1: Optimized Prime Checker
- Efficient prime number checking
- Uses square root optimization
- Skips unnecessary iterations for better performance

---

### 🔹 Level 2: Fibonacci Comparison
- Implements two approaches:
  - Naive recursive method
  - Optimized memoization method
- Compares execution time to highlight performance differences

---

### 🔹 Level 3: Advanced CLI Calculator
- Interactive command-line interface
- Supports:
  - Addition
  - Subtraction
  - Multiplication
  - Division
- Includes:
  - Error handling (e.g., division by zero)
  - Continuous execution loop
  - Operation history tracking
